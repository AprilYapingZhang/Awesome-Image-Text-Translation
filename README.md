- Cong Ma, Yaping Zhang, Mei Tu, Yang Zhao, Yu Zhou, Chengqing Zong. [E2TIMT: Efficient and Effective Modal Adapter for Text Image Machine Translation](https://arxiv.org/pdf/2305.05166.pdf)[C]. ICDAR 2023.
